# KGXplore 2026 - Digital Transformation Event Microsite (Concept Project)

A high-performance, interactive marketing landing page built as a concept project for KG Media (Kompas Gramedia). This project demonstrates the full workflow of a Hybrid Frontend Developer: from conceptualizing user flows, designing high-fidelity wireframes in Figma, to executing pixel-perfect, responsive code using semantic HTML5, modern CSS3, and vanilla JavaScript.

## 🎯 Project Overview & Business Goals
* **Target Department:** Creative Marketing (Advertising & Publisher Product).
* **Objective:** Create an engaging event microsite to drive registrations for the KGXplore 2026 Tech Exhibition.
* **Core Metrics:** Fast mobile loading speed (Core Web Vitals), interactive layout (Bento Grid), and built-in conversion tracking[cite: 1].

---

## 🎨 Phase 1: UX/UI Design Process

### 1. User Flow & Wireframing
Before writing any code, the user journey was mapped to maximize conversion rates (Event Registrations)[cite: 1]:
* **User Flow:** Landing Page ➔ Dynamic Event Schedule ➔ Register Button Click ➔ Floating Modal Form ➔ Interaction Log Triggered.
* **Wireframe Concept:** Low-fidelity sketches prioritized a mobile-first layout, ensuring key marketing banners and call-to-action (CTA) buttons sit firmly "above the fold."
